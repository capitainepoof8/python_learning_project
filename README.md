# Python Learning Project

Une collection de petits scripts Python écrits dans le cadre de mon apprentissage de la programmation en MIAGE (CESAG, Licence 1). Chaque fichier est un exercice indépendant qui couvre les bases du langage : entrées/sorties, calculs, conditions et manipulation de variables.

## 📂 Contenu du projet

| Fichier | Description |
|---|---|
| [`main.py`](main.py) | Demande le nom et l'année de naissance de l'utilisateur, calcule son âge et l'âge qu'il aura l'année suivante. |
| [`presentation.py`](presentation.py) | Petit script de présentation personnelle (nom, prénom, âge, filière). |
| [`moyenne.py`](moyenne.py) | Calcule la moyenne de trois notes saisies par l'utilisateur. |
| [`bulletin.py`](bulletin.py) | Génère un bulletin de notes pour 5 cours (devoir 40% / examen 60%) et calcule la moyenne générale. |
| [`imc.py`](imc.py) | Calcule l'Indice de Masse Corporelle (IMC) à partir du poids et de la taille. |
| [`convertisseur.py`](convertisseur.py) | Convertit un montant en Francs CFA vers l'Euro et le Dollar US. |
| [`cybercafe.py`](cybercafe.py) | Calcule le montant à payer pour une session dans un cybercafé, avec une réduction si la durée est inférieure à 3 heures. |
| [`tp2.py`](tp2.py) | Simule une facture pour "Sama Multi Services" (pain, lait, sucre) avec calcul du montant HT, de la TVA (18%) et du TTC. |

## 🚀 Utilisation

Chaque script est autonome et s'exécute directement avec Python 3 :

```bash
python3 nom_du_script.py
```

Par exemple, pour lancer le calculateur d'IMC :

```bash
python3 imc.py
```

Le programme demande ensuite les informations nécessaires via la console (`input()`).

## 🛠️ Prérequis

- Python 3.x installé sur votre machine
- Aucune dépendance externe : tous les scripts utilisent uniquement la bibliothèque standard

## 🎯 Objectif

Ce dépôt n'est pas un projet applicatif unique, mais un carnet d'exercices pratiques permettant de mettre en application :
- la saisie utilisateur (`input()`) et la conversion de types (`int()`, `float()`)
- les opérations arithmétiques et le formatage des résultats (`round()`)
- les structures conditionnelles (`if`)
- les calculs métiers simples (facturation, conversion de devises, moyennes scolaires)
